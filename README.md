# Suda Telegram Bot

**Suda Telegram Bot** — Telegram-бот для кофейни "Сюда"
Бот автоматизирует акцию лояльности: **"Купи 7 напитков — 8-й в подарок"**, и помогает бариста выдавать клиентам уникальные коды для получения очков.

---

## Функционал

### Для клиентов:
- **Регистрация** через ввод ФИО и номера телефона.
- **Получение уникального 6-значного цифрового кода** через кнопку **"Получить код"**.
- **Ввод кода** в бота для получения 1 очка.
- **Отслеживание прогресса**: сколько очков набрано и сколько осталось до бесплатного напитка.
- **Просмотр правил акции**.

### Для бариста:
- **Автоматический доступ** к панели управления при нажатии `/start`.
- **Выдача кода клиенту** по фамилии и последним 4 цифрам телефона.
- **Проверка баллов клиента** по фамилии и последним 4 цифрам телефона.
- **Просмотр правил акции**.
